# 💸 Group Expense Splitter (Splitwise Clone)

A web-based application to manage and split group expenses efficiently.  
It tracks shared expenses, calculates who owes whom, and provides both clear (pairwise) and optimized settlement methods.

## 🚀 Live Demo  
https://expense-splitter-aditi.streamlit.app

## 🎥 Demo Video  


## 💡 Features  
- Group creation and member management  
- Add and track shared expenses  
- Detailed breakdown of who owes whom  
- Pairwise settlement for clear understanding  
- Optimized settlement to minimize transactions  
- Interactive and user-friendly UI  

## ⚙️ How It Works  
- Records expenses with payer and participants  
- Calculates individual shares  
- Builds debt relationships between users  
- Generates pairwise settlement (direct balances)  
- Applies greedy algorithm for optimized settlement  

## 🛠 Tech Stack  
- Python  
- Streamlit  
- Data Structures (Dictionary-based debt tracking)  

## 🎯 Use Case  
Designed for managing shared expenses in groups such as trips, roommates, and events, enabling users to track payments, avoid confusion, and settle balances efficiently with clear or optimized transactions.
